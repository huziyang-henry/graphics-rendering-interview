# 新增内容检查清单

- [ ] YAML frontmatter 格式正确且字段完整
- [ ] difficulty 为四个枚举值之一
- [ ] knowledge_points 的 id 和 name 正确
- [ ] 五步结构完整，每步内容充实
- [ ] 数学公式使用 `$` 或 `$$` 包裹，无不兼容环境
- [ ] 单字母变量使用 Markdown 斜体而非 `$...$`
- [ ] 向量统一使用 `\mathbf{}`
- [ ] 行内公式前后有空格，块级公式前后有空行
- [ ] 文件命名使用英文连字符
- [ ] 对应章节的 index.md 已更新
- [ ] 无截断行（`\` 结尾）和 Unicode 转义残留
- [ ] 如有通用经验，已更新对应规范文件（先改 `.claude/docs/` 子文件，再合并到 Cursor/Copilot 版本）
